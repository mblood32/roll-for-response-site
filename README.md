# Roll for Response Website

This is a Vite + React deployment package for the Roll for Response landing page.

## Free hosting options
- **Vercel Hobby**: free for personal/pre-launch projects.
- **Netlify Free**: free forever for small experiments and previews.
- **GitHub Pages**: free static hosting from a GitHub repo.

## Best fit
- **Vercel**: easiest for this package and very smooth previews.
- **Netlify**: also a strong free option.
- **GitHub Pages**: free, but a little less convenient for React app setup.

## Quick start
```bash
npm install
npm run dev
npm run build
```

## Deploy to Vercel
1. Create a GitHub repo and upload these files.
2. Import the repo into Vercel.
3. Framework preset: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`

## Deploy to Netlify
1. Create a GitHub repo and upload these files.
2. Import the repo into Netlify.
3. Build command: `npm run build`
4. Publish directory: `dist`

## Deploy to GitHub Pages
Use a GitHub Actions workflow or a Pages-compatible Vite setup.
